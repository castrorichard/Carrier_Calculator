# 📱 Calculadora de Franquia de Operadora

Um programa simples em Java desenvolvido para praticar a lógica condicional e revisar a sintaxe de 
**operadores de atribuição cumulativa** (como o `+=`).

## 🎯 Objetivo do Código
O sistema calcula o valor final de uma conta de telefonia com base nos minutos de uso informados pelo usuário:
- **Franquia básica:** R$ 50,00 (limite de 100 minutos).
- **Minutos excedentes:** Cobrança de R$ 2,00 por minuto adicional.

Para o cálculo dos minutos extras, o código utiliza o operador `+=` para somar o custo adicional diretamente na 
variável da conta, garantindo uma sintaxe mais limpa e eficiente.

## 🛠️ Tecnologias Utilizadas
- **Java**
- `java.util.Scanner` (Para leitura de dados via console)
- `java.util.Locale` (Para formatação de moeda em padrão US)
